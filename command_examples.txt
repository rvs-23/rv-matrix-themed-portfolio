--- Matrix Terminal Portfolio: Command Examples ---

This file provides a comprehensive list of all available commands and how to use them.

--- Navigation & Information ---
help
    Description: Displays a list of all available commands and their brief descriptions.
    Example: help

about
    Description: Displays detailed information about the portfolio owner.
    Example: about

contact
    Description: Shows contact information (Email, LinkedIn, GitHub, Medium).
    Example: contact

whoami
    Description: Displays the current user's name (as configured).
    Example: whoami

date
    Description: Displays the current date and time.
    Example: date

projects
    Description: Shows featured projects (links to GitHub).
    Example: projects

skills
    Description: Lists key skills in a summary format.
    Example: skills

skilltree [path]
    Description: Explore a detailed, hierarchical tree of skills.
    Examples:
        skilltree
        skilltree se
        skilltree "Software Engineering"
        skilltree "ai > genai"
        skilltree "Artificial Intelligence & Machine Learning > Generative AI"

download cv
    Description: Downloads the CV (resume) of the portfolio owner or opens it in a new tab.
    Example: download cv

--- Terminal & Display ---
clear
    Description: Clears the terminal output, keeping the welcome message.
    Example: clear

theme <name|mode>
    Description: Changes the visual theme of the terminal and background.
    Available specific themes: amber, cyan, green (default), purple, twilight, crimson, forest.
    Available general modes: light, dark (reverts to default green).
    Examples:
        theme twilight
        theme light
        theme amber
        theme dark

resize term <width> <height>
    Description: Resizes the terminal window. Units can be px, %, vw, vh.
    Examples:
        resize term 70vw 60vh
        resize term 800px 600px
        resize term 90% 85%

resize term reset
    Description: Resets the terminal window to its default size.
    Example: resize term reset

--- Matrix Rain Configuration ---
rainconfig
    Description: Shows the current settings for the Matrix rain animation.
    Example: rainconfig

rainconfig <parameter> <value>
    Description: Sets a specific parameter for the Matrix rain animation.
    Parameters & Example Values:
        fontSize: 8-40 (e.g., rainconfig fontSize 12)
        speed: 20-500 (milliseconds, lower is faster, e.g., rainconfig speed 50)
        density: 0.1-3.0 (multiplier for columns, e.g., rainconfig density 0.8)
        trailEffect: true | false (e.g., rainconfig trailEffect false)
        randomizeSpeed: true | false (e.g., rainconfig randomizeSpeed true)
        fontFamily: string (e.g., rainconfig fontFamily "Courier New, monospace")
    Examples:
        rainconfig fontSize 10
        rainconfig speed 70
        rainconfig trailEffect false

rainconfig reset
    Description: Resets all Matrix rain settings to their default values.
    Example: rainconfig reset
